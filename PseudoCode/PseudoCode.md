# 의사코드 (Pseudo-Code)

### Syntax

-----

***Indentation*** 을 통해 확실한 영역을 구분지어줘야한다. (파이썬처럼)

* **Assign**  **←** 

  ex) var ← 1

  <br>

* **Logical Operation**

  * && — *and*
  * ||  —  *or*

  <br>

* **if else chain**

  ***if***  *condition*

  ​	***then***  *sequence1*

  ​			  *sequence2*

  ​			  ・・・・・・・

  ***else if***  *condition*

  ​	***then***  *sequence1*

  ​			  *sequence2*

  ​			  ・・・・・・・

  ***else***

  ​	***then***  *sequence1*

  ​			  *sequence2*

  ​			  ・・・・・・・

* **loop**

  * **for loop**

    ***for***  *start*  ***to***  *end*

    ​		***do***  *sequence1*

    ​			   *sequence2*

    ​			   ・・・・・・・

  * **while loop**

    ***while***  *condition*

    ​		***do***  *sequence1*

    ​			   *sequence2*

    ​			   ・・・・・・・

* **Length**

  A is kind of array or list

  * |A|
  * length[A]
